Media Harvester

Preview media names first, then download with layered fallback (direct, yt-dlp, format fallback) and debug reports.

Tool checklist

Use Tool check only to verify runtime support for Python, yt-dlp, ffmpeg, curl, aria2c, and gallery-dl.

  • yt-dlp: extraction + format selection
  • ffmpeg: merge/remux streams
  • curl: direct transfer fallback
  • aria2c: optional fast segmented downloader
  • gallery-dl: extractor for image/media galleries
Control tools

Help: controls explained

  • Preview limit: how many candidate items appear in the selector preview.
  • yt-dlp primary format: first format expression tried for each item.
  • Fallback formats: extra format expressions tried in order when the primary fails.
  • List formats: prints available source formats so you can troubleshoot resolution failures.
  • Debug mode: stores a media_harvester_debug.json report with failures + environment status.
  • Tool check only: prints a readiness checklist and install hints.
