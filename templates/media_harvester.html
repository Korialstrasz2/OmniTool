Passive stream capture (experimental)

Use this when a site only reveals stream URLs during playback. Slow mode captures passively while the video plays; fast mode tries to fetch the full stream from intercepted URLs as soon as candidates appear.

  1. Enter the page URL that contains the media player.
  2. Choose mode: Slow (observe playback) or Fast (aggressive full fetch after interception).
  3. Set capture window and idle timeout, then run capture.
  4. Review output logs to see intercepted stream URLs and download results.
