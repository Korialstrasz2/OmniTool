Help: controls explained

  • Workers: number of parallel downloads. Increase for faster connections, lower for unstable networks.
  • Run mode: preview, standard fallback download, or all-resolutions download.
  • Diagnostics output: prints format probe + troubleshooting logs so you can see why a resolution failed.
  • Include page URL with yt-dlp: useful when direct media links are hidden but extractor works on page URL.
  • Force all resolutions: enumerates yt-dlp formats and attempts one file per discovered resolution.
  • Install / Upgrade Dependencies: installs/upgrades supported downloader dependencies.
