Voxtral Local Suite (STT/TTS)

Setup locale orientato alla privacy per pipeline STT/TTS con ricerca GGUF, download modelli, opzioni di voice cloning (speaker reference), e integrazione hotkey (Windows AutoHotkey / script Linux). Il tool usa solamente esecuzione locale: nessuna chiamata cloud obbligatoria.

Configurazione globale

Percorsi persistenti, modelli default e parametri runtime base.

Bootstrap locale (venv + dipendenze)

Crea virtualenv dedicato e installa dipendenze STT/TTS; abilita GPU e clone voice dove richiesto.

GGUF scouting + download modelli

Ricerca modelli su Hugging Face (query libera) e scarica file manualmente in path locale.

STT professionale

Trascrizione locale con VAD, timestamps parola-per-parola, beam search e controllo device.

TTS + voice cloning

Genera voce locale (multilingua) con reference speaker opzionale per cloning one-shot.

Automazione esterna (hotkey)

Genera launcher AutoHotkey/Script per richiamare STT/TTS fuori da OmniTool.
